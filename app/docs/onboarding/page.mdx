# Onboarding guide

## Objectives
The objective of this onboarding guide is to help new members quickly become familiar with the development process of OverCoded.

## Software Prerequisites
This list is recommended but not mandatory. Feel free to use any software you want.

- Docker
- Postman
- Terminal (MacOS or Linux)

Backend:
- IntelliJ IDEA Ultimate
- Java 17/21
- Maven

Web Frontend:
- Node v22+
- npm

Mobile:
- Android Studio
